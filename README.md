# Claudicus

A top-down 2D tile-based sprite RPG set in a gritty medieval fantasy world. Built with TypeScript and HTML5 Canvas — runs entirely in the browser with no dependencies.

## Setting

The rural outskirts of a small feudal kingdom. You are a peasant-turned-militia member defending your village of Brannford from bandits, wolves, and darker things lurking in Thornwood forest.

## Tech

- **TypeScript** — vanilla, no frameworks
- **HTML5 Canvas** — all sprites drawn in code (no image assets)
- **localStorage** — save/load game state
